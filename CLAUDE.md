# Claude instructions for bletracking

## After every session
Update `CONTEXT.md` before ending the conversation:
1. Update **Current state** to reflect what was added/changed
2. Append a new dated entry to the **Session log** section describing what was done
3. Update the **File map** if any files were added or removed
4. Update **Known limitations / future work** if items were resolved or new ones identified
